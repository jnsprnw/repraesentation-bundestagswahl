# Repräsentation der Bundestagswahl

Die Ergebnisse der Bundestagswahl 2025 repräsentiert nicht die tatsächliche Bevölkerung. Hier wird gezeigt, wie sich die Stimmenanteile je nach verwendeter Gesamtmenge verändern. Inspiration und Daten von [Arne Semsrotts](https://www.instagram.com/arne.semsrott/) [Vortrag](https://www.youtube.com/watch?v=7waHkzEHcuw) bei der re:publica 26. Basierend auf Ergebnissen der ersten Hochrechnung.

## Developing

Once you've created a project and installed dependencies with `bun install`, start a development server:

```sh
npm run dev

# or start the server and open the app in a new browser tab
npm run dev -- --open
```

## Building

To create a production version of your app:

```sh
npm run build
```
